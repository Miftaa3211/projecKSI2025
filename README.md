# projecKSI2025
# Analisis dan Implementasi Sistem Pendaftaran Wisuda Digital

Proyek ini dikembangkan untuk :
1. menggantikan sistem pendaftaran wisuda manual yang masih digunakan di Politeknik Negeri Lampung. 
Sebelumnya, proses pendaftaran dilakukan melalui formulir kertas sehingga sering menimbulkan kendala seperti antrian panjang, kesalahan data, dan proses validasi yang memakan waktu lama. Melalui sistem ini, seluruh proses pendaftaran diubah menjadi berbasis web agar lebih cepat, efisien, dan mudah digunakan oleh mahasiswa maupun pihak kampus.

2. Sistem ini dibangun menggunakan Node.js sebagai backend dan MySQL sebagai basis data untuk mengelola informasi mahasiswa serta proses verifikasi secara otomatis. Sementara itu, tampilan antarmuka dirancang menggunakan Figma agar lebih modern, rapi, dan mudah dipahami pengguna.

Hasil analisis menunjukkan bahwa sistem digital ini mampu menurunkan tingkat keterlambatan pendaftaran dari 22% menjadi 8%, serta mempercepat waktu verifikasi. 

Melalui sistem ini, proses pendaftaran wisuda menjadi lebih transparan, efisien, dan terdata dengan baik. Mahasiswa dapat mendaftar secara online, memantau status verifikasi, serta menerima notifikasi saat data mereka disetujui. Bagi pihak kampus, data yang terkumpul bisa langsung digunakan untuk analisis tanpa perlu proses input manual tambahan.
